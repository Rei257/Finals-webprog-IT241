# Project Screenshots

Place your project screenshots in this folder.

## How to use

Add your project images here and name them to match the projects:

- `midterm-project.png` — Screenshot of your midterm project
- `flutter-app.png` — Screenshot of your Flutter app
- `project3.png` — Screenshot of your third project

Supported formats: `.jpg`, `.png`, `.webp`

The project cards will automatically display these images.
